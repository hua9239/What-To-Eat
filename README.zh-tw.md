## 建置

1. 將專案複製到本地端
    ```bash
    git clone https://github.com/hua9239/What-To-Eat.git
    ```
2. 進入專案目錄
    ```bash
    cd What-To-Eat
    ```
3. 編譯專案
    ```bash
    g++ -o main.exe -I .\include\ .\src\Food.cpp .\src\main.cpp .\src\Restaurant.cpp
    ```
